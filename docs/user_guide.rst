User Guide
==========

This guide provides in-depth instructions for using CYCLOPS to simulate and analyze Heme-Ligand complexes.

.. contents:: Table of Contents
   :local:
   :depth: 2

Standard Workflow
-----------------

The standard CYCLOPS workflow consists of three main stages:

1.  **Preparation:** Cleaning PDBs, fixing structural gaps (RFAA), and protonating ligands.
2.  **Simulation:** Parameterizing the Heme-Ligand complex and running Molecular Dynamics.
3.  **Analysis:** Extracting stability metrics and interaction profiles (PLIP).

For a basic example of running a simulation, please see the :doc:`getting_started`.

Structure Repair (RFAA Fixer)
-----------------------------

If you are working with structures generated by **RoseTTAFold All-Atom (RFAA)**, you may encounter structural gaps or "hybrid" residues (where a ligand and a residue are merged).

CYCLOPS includes a tool to automatically repair these files:

.. code-block:: python

    from cyclops.rfaafixer import RFAAFixer

    fixer = RFAAFixer("rfaa_generated.pdb")
    fixer.split_and_repair("repaired_output.pdb")

This will:

* Detect the structural gap near the Iron/Heme site.
* Split hybrid residues (e.g., ``LG1``) into their components (e.g., ``CYS`` + ``HEM``).
* Renumber the sequence to close the gap.

Analysis Tools
--------------

After your simulation is complete, you can use the analysis modules to check stability and interactions.

**1. Heme Stability Analysis**

.. code-block:: python

    from cyclops.analysis import HemeAnalysis

    analysis = HemeAnalysis(
        simulation_path="my_simulation",
        ligand_id="UNL"
    )
    results = analysis.run_all_analyses()

**2. PLIP Interaction Profiling**

To track how protein-ligand interactions evolve over the trajectory:

.. code-block:: python

    from cyclops.plipanalyser import MDPlipAnalysis

    analyzer = MDPlipAnalysis(
        pdb_id="1TQN",       # Reference PDB
        ligand_code="HEM",
        working_dir="my_simulation"
    )
    
    # Analyze the trajectory
    df = analyzer.step2_analyze_trajectory(
        trajectory_file="my_simulation/trajectory_complex.xtc",
        topology_file="my_simulation/topology_complex.pdb"
    )
    print(df.head())
